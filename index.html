One-time Firebase setup

Get your Firebase config

If you haven't created a Firebase project yet, follow these steps first:

Go to console.firebase.google.com and create a new project. Skip Google Analytics.
In the sidebar, click Build → Realtime Database. Click Create database, pick any location, select Start in test mode.
Click the ⚙️ gear icon → Project settings. Scroll to "Your apps," click </> Web, register an app (any name). Copy the entire firebaseConfig = { … } block shown.
